## 🚀 Запуск проекта

### Быстрый старт

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/Danikpol/YBM.org.git
cd YBM.org
```

2. **Откройте в браузере:**
```bash
# Просто откройте index.html в любом браузере
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

3. **Или используйте локальный сервер:**
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server -p 8000

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.

### Развертывание на GitHub Pages

Проект автоматически развертывается на GitHub Pages при пуше в ветку `main` или `master`.

**URL:** https://danikpol.github.io/YBM.org/
